重新定义解析体验

多源抽象

ParseSource<T> 仅暴露 getSource()；ByteSource、CharSource、ObjectSource 各自提供字节/字符/对象读方法。

组合子 API

optional、many、choose、repeat、map、peek 等，自由组合出复杂解析逻辑，无需手写状态机。

位置与回退

currentPosition() / setCurrentPosition(int)，解析失败时可回退；optional 在 ReadException 时自动回退并返回 null。

对象源与 Scope

fromObject(root)、enterScope / exitScope，用于层级对象解析与 OGNL 表达式取值。

快速示例

import static com.github.ocaso1987.eater.Parsers.*;

ParseContext ctx = ParseContext.fromChars("hello world");
String first = exactString("hello").parse(ctx);  // "hello"
String rest  = exactString(" world").parse(ctx); // " world"
